# Minishell

## Features

### Builtin
- ``cd``
- ``pwd``
- ``echo``
- ``exit``
- ``export``
- ``unset``
- ``env``

### Signal
- ``CTRL-C``
- ``CTRL-\``
- ``CTRL-D``

### Redirection
- ``>``
- ``>>``
- ``<``

### Pipe
- ``|``

### Semicolons
- ``;``

### Environment Path
- ``$?``
- ``:``
- ``=``
- env expansions + ``$?``
